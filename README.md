## Namenskonvention für Objekte

Alle Objekte sollen mit dem Kürzel **ZLJND(2)** beginnen:

- **Z** ist Pflichtbestandteil
- **LJND** steht für: Laurenz, Jonas, Niklas, Dominik
- **2** für die 2. App enthalt das Kürzel noch eine 2

### Tabellen

- `ZLJND_EMPLOYEE`
- `ZLJND_VAC_REQ`
- `ZLJND_VAC_ENT`

### Root Views

- `ZLJND_C_EMPLOYEE`
- `ZLJND2_C_VAC_REQ`
